# SQL Business Case Studies

## Project Overview

This repository contains a collection of SQL case studies based on real-world business scenarios from different industries, including streaming services, online education, banking, and ride-hailing platforms.

The projects demonstrate how SQL can be applied to solve analytical problems, prepare datasets, evaluate business hypotheses, and generate insights that support business decision-making.

---

## Technologies

* PostgreSQL
* SQL
* Window Functions
* Common Table Expressions (CTE)
* JOIN
* CASE
* Aggregate Functions
* Date & Time Functions

---

## Business Cases

### Customer Purchase Analysis (Streaming Service)
Identified first and repeat customer purchases using window functions and analyzed acquisition partner performance.

### Revenue Trend Analysis (Online Education)
Analyzed monthly revenue and customer activity, applying moving averages to identify long-term trends.

### Airport Queue Analysis (Ride-Hailing Service)
Evaluated driver waiting times and order success rates to determine the optimal arrival time at the airport.

### Feature Engineering for Credit Scoring (Banking)
Prepared an analytical dataset for a machine learning model by creating customer-level features and aggregated metrics.
